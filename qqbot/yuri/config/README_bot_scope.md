# Bot启用范围管理功能

## 功能说明

该功能允许你指定bot对哪些群聊和好友启用，支持白名单和黑名单两种模式。

## 配置文件

配置文件位于：`/home/jianying/code/bot/qqbot/yuri/data/bot_scope.json`

首次运行时会自动创建默认配置，包含以下内容：
- 默认启用的群聊：284205050, 908188794
- 默认启用的用户：2193807541, 1185329732
- 默认管理员：2193807541, 1185329732
- 默认模式：白名单模式（whitelist）

## 使用方法

### 查看当前配置
```
/bot范围
```
显示当前的启用范围配置和管理员列表。

### 添加群聊到启用列表
```
/添加群 <群号>
```
例如：`/添加群 123456789`

### 从启用列表移除群聊
```
/移除群 <群号>
```
例如：`/移除群 123456789`

### 添加用户到启用列表
```
/添加用户 <用户ID>
```
例如：`/添加用户 123456789`

### 从启用列表移除用户
```
/移除用户 <用户ID>
```
例如：`/移除用户 123456789`

### 设置启用模式
```
/设置模式 <whitelist/blacklist>
```
- `whitelist`：白名单模式，只有在列表中的群聊/用户才能使用bot
- `blacklist`：黑名单模式，列表中的群聊/用户不能使用bot

例如：`/设置模式 blacklist`

### 添加管理员
```
/添加管理员 <用户ID>
```
例如：`/添加管理员 123456789`

### 移除管理员
```
/移除管理员 <用户ID>
```
例如：`/移除管理员 123456789`

## 权限说明

- 只有管理员才能使用以上所有命令
- 管理员默认包括：2193807541, 1185329732
- 管理员可以添加或移除其他管理员

## 影响范围

该功能影响以下插件：
1. **签到插件**（sign_in.py）：只有在启用范围内的用户才能签到
2. **AI聊天插件**（yuri_bot.py）：只有在启用范围内的用户才能使用AI聊天功能

## 配置示例

### 白名单模式示例
```json
{
  "mode": "whitelist",
  "enabled_groups": [284205050, 908188794],
  "enabled_users": [2193807541, 1185329732],
  "disabled_groups": [],
  "disabled_users": [],
  "admin_users": [2193807541, 1185329732]
}
```
在此模式下，只有在 `enabled_groups` 中的群聊或 `enabled_users` 中的用户才能使用bot。

### 黑名单模式示例
```json
{
  "mode": "blacklist",
  "enabled_groups": [],
  "enabled_users": [],
  "disabled_groups": [123456789],
  "disabled_users": [987654321],
  "admin_users": [2193807541, 1185329732]
}
```
在此模式下，除了 `disabled_groups` 中的群聊和 `disabled_users` 中的用户外，其他所有群聊和用户都能使用bot。

## 注意事项

1. 配置会自动保存到JSON文件，重启后仍然有效
2. 在白名单模式下，`disabled_groups` 和 `disabled_users` 会被忽略
3. 在黑名单模式下，`enabled_groups` 和 `enabled_users` 会被忽略
4. 建议使用命令来修改配置，而不是直接编辑JSON文件
